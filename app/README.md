## 📖 About

**PlaceAI** is a full-stack AI-driven placement preparation platform that bridges the gap between coding practice and real-world interview readiness. It combines **AI-powered interview simulations**, **human expert mock interviews**, **adaptive aptitude analytics**, and **ATS resume analysis** into a single, unified platform.

Built with **Next.js 16 (App Router)**, **React 19**, **TypeScript**, and powered by **SambaNova AI (Meta-Llama models)**, it delivers a modern dark-themed glassmorphism UI with role-based access for **Students** and **Industry Experts**.

---

## ✨ Features

### 🤖 AI Interview Simulation
- Personalized interview questions generated by **Meta-Llama-3.3-70B-Instruct** based on your resume skills, target role, interview type, and difficulty level
- Multi-phase interview flow: **Loading → Fullscreen → Coding → Voice → Follow-up → Results**
- Live code editor with **Monaco Editor** (Python, JavaScript, Java, C++)
- Simulated code execution with mock test case results, runtime, and memory stats
- Voice-based AI evaluation using **Web Speech API** (speech-to-text)
- Real-time scoring across 4 metrics: **Technical (40%)**, **Problem Solving (25%)**, **Communication (20%)**, **Optimization (15%)**

### 🛡️ Anti-Cheat System
- Fullscreen enforcement at interview start
- Tab switch detection with violation counter
- Copy-paste monitoring
- Session flagging at 3+ violations with warning banners

### 📄 Resume Upload & ATS Analysis
- Drag-and-drop resume upload (PDF, PNG, JPG, JPEG, WEBP — max 10MB)
- **PDF resumes** analyzed by **Meta-Llama-3.3-70B-Instruct**
- **Image resumes** analyzed by **Llama-3.2-11B-Vision-Instruct** (Vision model)
- Detailed ATS Score (0–100) with grade (A/B/C/D) and breakdown:
  | Category | Max Points |
  |---|---|
  | Skills & Keywords | 30 |
  | Sections Completeness | 25 |
  | Action Verbs & Impact | 15 |
  | Contact Information | 15 |
  | Content Density | 15 |
- Technical Score & Communication Score (0–100 each)
- Auto-extracted skills added to your profile
- AI-generated role recommendations with match percentages
- Actionable suggestions (good / warning / error)
- Results cached by file hash in SQLite for instant repeat lookups

### 📝 Timed Aptitude Quizzes
- **3 Sections**: Quantitative Aptitude, Logical Reasoning, Verbal Ability
- 10 questions per section with 4 MCQ options each
- Countdown timer with auto-submit (visual warning under 60 seconds)
- Question flagging for later review
- Color-coded navigation grid (answered / flagged / not visited)
- Detailed results with per-question review and explanations

### 👨‍💼 Expert Mock Interview Booking
- **4-step booking wizard**: Select Role → Choose Expert → Pick Schedule → Confirm
- 4 expert profiles (from Google, Amazon, Microsoft, Flipkart) with bios, expertise tags, ratings
- 3-day calendar with hourly time slots
- Student's full profile, scores, and weak areas shared with expert

### 👩‍🏫 Expert Dashboard & Availability
- Manage availability across 3-day calendar with 12 hourly slots (9 AM–8 PM)
- View upcoming interviews with student profiles
- Post-interview student feedback form (1–5 star rating + text)
- Session stats: total sessions, monthly count, average rating, positive review %

### 📊 Analytics & Performance Tracking
- **Radar chart**: 6-axis skill web (Quantitative, Logical, Verbal, Technical, Communication, Problem Solving)
- **Area chart**: Interview score trend over time
- Section-wise accuracy table with best/average scores, attempts, and trends
- Identified weak areas with priority levels (High / Medium)
- AI-generated personalized improvement plan (5 recommendations)
- Placement Readiness score

### 👤 Profile Management
- 4-tab profile editor: Basic Info, Education, Skills, Links & Resume
- 27 quick-add popular skill suggestions
- Manual skill adding/removing
- LinkedIn, GitHub, and resume links
- Skills feed into AI interview personalization

### 🔐 Role-Based Access Control
- **Student**: Dashboard, Profile, Aptitude Quiz, AI Interview, Book Expert, Analytics
- **Expert**: Dashboard, Availability Management, Bookings
- Quick demo login with one-click access for Student and Expert roles

## 🚀 Getting Started

### Prerequisites

- **Node.js** 18+ installed
- **npm** or **yarn** package manager
- **SambaNova API Key** (for AI features)

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/Vinitcharde/Enigma2.0-the-dyno.git

# 2. Navigate to the app directory
cd Enigma2.0-the-dyno/app

# 3. Install dependencies
npm install

# 4. Create environment file
echo "SAMBANOVA_API_KEY=your_api_key_here" > .env.local

# 5. Start the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Environment Variables

| Variable | Required | Description |
|---|---|---|
| `SAMBANOVA_API_KEY` | Yes | API key from [SambaNova](https://sambanova.ai/) for resume parsing & interview question generation |

## 🗄️ Database Schema

SQLite database (`placeai.db`) is auto-created on first run:

```sql
-- Resume analysis cache & history
CREATE TABLE resume_analyses (
    id                  INTEGER PRIMARY KEY AUTOINCREMENT,
    user_email          TEXT NOT NULL,
    filename            TEXT NOT NULL,
    file_hash           TEXT NOT NULL,       -- MD5 hash for cache deduplication
    analyzed_at         DATETIME DEFAULT CURRENT_TIMESTAMP,
    ats_score           INTEGER,
    grade               TEXT,                -- A / B / C / D
    technical_score     INTEGER,
    communication_score INTEGER,
    skills              TEXT,                -- JSON array
    suggested_roles     TEXT,                -- JSON array
    full_analysis       TEXT                 -- Full JSON blob
);

-- Per-user extracted skills
CREATE TABLE user_skills (
    id         INTEGER PRIMARY KEY AUTOINCREMENT,
    user_email TEXT NOT NULL,
    skill      TEXT NOT NULL,
    added_at   DATETIME DEFAULT CURRENT_TIMESTAMP,
    UNIQUE(user_email, skill)
);
```

## 🔌 API Endpoints

### `POST /api/parse-resume`

Analyzes an uploaded resume using AI and returns ATS scoring with extracted data.

**Request**: `multipart/form-data`
| Field | Type | Description |
|---|---|---|
| `resume` | File | PDF or image file (max 10MB) |
| `email` | String | User's email for caching |

**Response**: ATS score, grade, breakdown, suggestions, extracted info (name, email, skills, experience, suggested roles)

---

### `POST /api/generate-questions`

Generates personalized interview questions based on user profile.

**Request**: `application/json`
| Field | Type | Description |
|---|---|---|
| `skills` | String[] | User's skill set |
| `role` | String | Target job role |
| `interviewType` | String | full / dsa / behavioral |
| `difficulty` | String | easy / medium / hard |
| `userName` | String | User's name |

**Response**: Array of question objects with `id`, `question`, `type`, `topic`, `hint`, `followUp`, `codeStarter`
